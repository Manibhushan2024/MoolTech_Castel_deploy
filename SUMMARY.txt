  API ENDPOINT


POST /api/enquiry

Accepts FormData with:
- fullName (string)
- phone (string)
- email (string)
- serviceArea (string)
- address (string)
- requestType (string)
- productType (string)
- description (string)
- voiceNote (file, optional)

Returns JSON:
{
  "success": true/false,
  "message": "...",
  "enquiry": { ... } // when successful
}

Server logs all enquiries to console.log()

  TROUBLESHOOTING


Build fails?
  npm install
  npm run build

Port 3000 in use?
  npm run dev -- -p 3001

Dark mode not working?
  Clear localStorage: localStorage.removeItem("theme")
  Refresh browser

Form not submitting?
  Check browser console for errors
  Verify /api/enquiry endpoint exists

Voice recorder not working?
  Check browser microphone permissions
  Fallback to file upload should work
